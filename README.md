#  AI Interview Coach

An AI-powered Interview Preparation Platform built using **Next.js**, **TypeScript**, **Tailwind CSS**, **Prisma**, and **OpenAI** to help students prepare for technical interviews.

The platform allows users to generate AI interview questions, analyze resumes, practice mock interviews, and track their interview preparation progress through an intuitive dashboard.



#  Features

-  AI Technical Interview Question Generator
-  Resume Analyzer
-  Mock Interview Preparation
-  Dashboard with Interview Statistics
-  User Login & Authentication UI
-  Pricing Plans Page
-  Interview Performance Tracking
-  Modern Responsive UI
-  Dark Theme

#  Installation

Clone the repository

bash
git clone https://github.com/your-username/AI-Interview-Coach.git


Go to the project folder

bash
cd AI-Interview-Coach

Install dependencies

bash
npm install

Create a `.env` file

env
DATABASE_URL="file:./dev.db"
OPENAI_API_KEY=your_openai_api_key


Generate Prisma Client

bash
npx prisma generate


Run the development server

bash
npm run dev


Open your browser


http://localhost:3000
